# CloudScope

## Multi-Cloud Infrastructure Monitoring Platform

CloudScope is a centralized monitoring platform that collects infrastructure metrics from AWS, Azure, and Google Cloud using Prometheus and visualizes them in Grafana.

## Tech Stack

- Docker
- Prometheus
- Grafana
- Alertmanager
- Node Exporter
- AWS
- Azure
- Google Cloud
- Terraform
- Ansible
